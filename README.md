# nextjs-todo-app

Next.jsの基礎練習(ToDoアプリ)

## デモ

https://nextjs-todo-app-phi-one.vercel.app/

## 機能

タスクの追加
タスクの削除
タスクの完了/未完了の切り替え
タスクのブラウザ保存(ページをリロードしてもデータがlocalStorageに残る)
フィルター機能(全て/未完了/完了済み)
タスク名の編集(タスクをダブルクリックで編集)
優先度分け機能(優先度：高/中/低)
並び替え機能(追加順/優先度順/ユーザー設定)
ドラッグ＆ドロップによる並び替え(@hello-pangea/dndを使用)

### 起動方法

```bash
npm install
npm run dev
```

ブラウザで http://localhost:3000 を開く。
